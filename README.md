tsar module for php-fpm
=======================

1. Install tsar && tsardevel

    $ git clone git://github.com/kongjian/tsar.git
    $ cd tsar
    $ make
    $ sudo make install
    $ sudo make tsardevel

1. Install tsar-php-fpm

    $ git clone https://github.com/guoxiaod/tsar-php-fpm.git
    $ cd tsar-php-fpm 
    $ make
    $ sudo make install
